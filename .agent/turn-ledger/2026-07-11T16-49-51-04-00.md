## Source-backed finding

`update()` captures `Object.values(state.units)` and then calls `updateUnit()` for each captured object. `damage()` can delete a target from `state.units`, but the deleted object remains in the captured array. There is no liveness check at `updateUnit()` entry, so a killed unit can still act later in the same tick.
