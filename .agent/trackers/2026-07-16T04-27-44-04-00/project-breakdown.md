## Summary

PhantomCommand is a static browser campaign game with a procedural Canvas2D menu, menu-owned WebAudio, fixed-step grave-ring combat, Canvas2D world/HUD/minimap rendering, WebGL CRT presentation, local persistence, source checks, static build and Pages delivery. Both routes continuously project time-varying motion, but neither observes `prefers-reduced-motion`, stores a product motion override, classifies essential versus ornamental animation, or publishes a reduced-motion result and visible-frame acknowledgement.
